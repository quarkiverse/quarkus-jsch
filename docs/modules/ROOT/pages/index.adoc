== Installation

If you want to use this extension, you need to add the `io.quarkiverse.jsch:quarkus-jsch` extension first.
In your `pom.xml` file, add:

[source,xml]
----
<dependency>
    <groupId>io.quarkiverse.jsch</groupId>
    <artifactId>quarkus-jsch</artifactId>
</dependency>
----
